Disjoint-set data structures for JavaScript.
See [docs](https://make-github-pseudonymous-again.github.io/js-disjoint-set).
Parent is
[@aureooms/js-data-structures](https://github.com/aureooms/js-data-structures).

```js
find( a ) === find( b ) ; // false
union( A , B ) ;
find( a ) === find( b ) ; // true
```
